# 🎓 برنامه طراحی آزمون - نیکزاد فرد

یک برنامه کامل برای طراحی آزمون با قابلیت خروجی PDF و Word و ساخت APK اندروید.

## ✨ ویژگی‌ها

- طراحی ۶ نوع سوال: صحیح و غلط، جای خالی، جواب کوتاه، تستی، جور کردنی، تشریحی
- خروجی PDF
- خروجی Word (DOCX)
- پیش‌نمایش زنده آزمون
- ساخت خودکار APK برای اندروید

## 📱 نحوه دریافت APK

1. به تب **Actions** در بالای این صفحه بروید
2. روی workflow با نام **Build Android APK** کلیک کنید
3. روی دکمه **Run workflow** کلیک کنید (اگر بیلد شروع نشده)
4. بعد از اتمام بیلد (۳-۵ دقیقه)، در بخش **Artifacts** فایل `exam-maker-apk` را دانلود کنید
5. فایل `app-debug.apk` داخل آن را روی گوشی اندرویدی نصب کنید
